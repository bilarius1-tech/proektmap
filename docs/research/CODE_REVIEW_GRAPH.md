## Ключевые метрики

- **82x** медианная экономия на 6 реальных репозиториях

- **0.7ms** поиск по графу из 27 000 рёбер

## Что взяли

### Концепт графа знаний
- AST (Tree-sitter) → узлы (функции, классы) + рёбра (вызовы, наследование)
- Blast-radius: изменение функции → все кто её вызывает + их тесты
- Hub-узлы: самые связанные (архитектурные хотспоты)
- Bridge-узлы: chokepoints
- Knowledge gaps: критические узлы без тестов
- Инкрементальное обновление: < 2 сек для 2900 файлов

### Почему это важно для вайбкодеров
AI-агент не должен читать ВЕСЬ проект каждый раз.
Граф = карта. AI читает только нужное.
Меньше токенов → быстрее → дешевле.

- Новый этап «Граф знаний» → 4 решения
- Концепт, blast-radius, практическое применение, авто-эволюция навыков
