## Что взяли

### 20 принципов дизайна
Не правила — «линзы» через которые смотреть на дизайн:

1. Иерархия — важное крупнее
2. Воздух — отступы = дорого
3. Консистентность — одинаковое = одинаково
4. Контраст — тёмное на светлом
5. Близость — связанное рядом
6. Выравнивание — сетка
7. Ограничение — меньше цветов, меньше шрифтов
8-20. Ритм, Баланс, Масштаб, Цвет, Типографика, Сетка, Иконки, Изображения, Анимация, Доступность, Адаптив, Производительность, Эмоция

### 5D-рубрикатор
1. **Эстетика** — нравится? цвета гармоничны?
2. **Доступность** — всем видно? контраст 4.5:1? alt у картинок?
3. **Производительность** — быстро грузится? Lighthouse > 90?
4. **Масштабируемость** — добавить блок без перевёрстки?
5. **Консистентность** — всё в одном стиле?
