# План контента: Этап 1 «Инструменты»

## Философия этапа

Не «вот тебе VS Code», а «выбери свой инструмент из лучших».

У каждого вайбкодера свой стиль. Кто-то любит Cursor, кто-то SourceCraft от Яндекса, кто-то VS Code с Copilot. Мы показываем ВСЕ варианты и помогаем выбрать.

## Решения

### 1. Выбрать AI-редактор кода
**Проблема:** В чём писать код с AI-помощником
**Варианты:**
- VS Code + продолжения (бесплатно, база)
- Cursor (лучший AI-автокомплит, $20/мес)
- SourceCraft (Яндекс, бесплатно, YandexGPT)
- OpenCode (терминал, open-source)

**Рекомендация:** Начни с VS Code + бесплатные расширения. Если нужно мощнее → Cursor.

### 2. Выбрать AI-платформу
**Проблема:** Какой AI использовать для генерации кода
**Варианты:**
- ChatGPT (универсальный, $20/мес)
- DeepSeek (бесплатно, код)
- Claude (лучший для архитектуры)
- OpenRouter (один ключ → все модели)

**Рекомендация:** DeepSeek (бесплатно) для старта. OpenRouter для гибкости.

### 3. Установить Node.js
**Проблема:** Среда для запуска JavaScript
**Рекомендация:** Node.js 20 LTS

### 4. Установить Git
**Проблема:** Сохранение версий кода
**Рекомендация:** Git + GitHub Desktop (визуально)

### 5. Настроить Reasonix
**Проблема:** AI-агент для терминала
**Рекомендация:** Установить, настроить контекст проекта
