# Кейсы — мост между каталогом и действием

## Философия

### Проблема рынка

Все AI-каталоги выглядят одинаково: списки инструментов, агентов, MCP-серверов.

Новичок видит 500 позиций и думает: «И что дальше?»

### Решение ProektMap

**Маршрут создания цифрового продукта:**

```
Идея → Кейс → Архитектура → MCP → Агенты → Стек → Деплой → Монетизация
```

Кейс — это мост между «хочу» и «делаю». Человек приходит с мыслью «Хочу AI-консультанта» и видит готовый разбор: как другие сделали, какой стек, сколько времени, какие ошибки.

### Уникальная позиция

ProektMap = **не каталог инструментов, а карта создания цифровых продуктов.**

## Модель данных

| sortOrder | Int | Порядок |

## Страницы

### `/cases` — витрина кейсов

- Карточки: название, сложность, время, доход, стек (иконки)
- Фильтры: уровень, время сборки, доход
- Сортировка: новинки / популярные / для новичков

### `/cases/[slug]` — детальный разбор

- Полный текст с Markdown
- Архитектурная схема (React Flow)
- Связанные MCP-серверы и агенты
- Блок «Типичные ошибки»
- Кнопка «Повторить» → запускает Blueprint

### Интеграция в Blueprint

- На главной странице Blueprint'а: «Похожие кейсы»
- В этапе «Инструменты»: ссылка на кейс

## 3 уровня кейсов

### Уровень 1: Новичок (1-2 дня)

- Telegram Bot
- AI Чат-виджет
- Генератор статей

### Уровень 2: Практик (1 неделя)

- CRM
- SEO сервис
- Автоворонка

### Уровень 3: Профи (1 месяц)

- SaaS продукт
- AI агентная система
- Маркетплейс

## Порядок реализации (завтра)

### Фаза 1: Модель + API (30 мин)
- Prisma модель CaseStudy

- Карточки с фильтрами

- Блок «Похожие кейсы» на главной Blueprint'а

## Контент для первого запуска (3 кейса)

### 1. AI SEO Аудитор
- **Стек:** Next.js + OpenAI + Firecrawl + Supabase
- **Время:** 2 дня
- **Доход:** 49$/мес
- **Шаги:** Интерфейс → Firecrawl → Аудит → БД → PDF → Stripe

### 2. AI Консультант для сайта
- **Стек:** Next.js + OpenAI + Telegram Bot + Supabase
- **Время:** 3 дня
- **Доход:** 99$/мес
- **Шаги:** Чат-виджет → Telegram CRM → База знаний → Подписка

### 3. Telegram Support Bot
- **Стек:** Node.js + Telegram API + OpenAI + Supabase
- **Время:** 1 день
- **Доход:** 29$/мес
- **Шаги:** BotFather → Обработчик → AI-ответы → Админка

## Правила для агентов

1. Вся разработка на русском языке
2. Использовать существующие компоненты (React Flow, Prisma, CSS-переменные)
3. Стиль: Swiss Design, плотная сетка, без скруглений
4. Иконки: Lucide (локально)
5. Шрифты: Montserrat (заголовки) + Inter (текст)
6. Тёмная тема: использовать `var(--color-bg-primary)`, не `white`
7. SSH: `/c/Windows/System32/OpenSSH/ssh.exe root@109.196.165.106`
8. Проект: `/var/www/www-root/data/www/proektmap.ru`
9. Сборка: `rm -rf .next && npx next build`
10. Деплой: `pm2 restart proektmap`
